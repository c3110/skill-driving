# 通过人行横道线

1. 没有语音播报!!需要自行观察道路情况
2. 进入人行横道线前主动刹车减速到每小时30公里以内
3. 注意礼让行人
4. 进入停止线前就减档到二档，车速控制在25公里左右最好，不要超过30公里
5. 任何一个动作或指令都要先观察道路交通情况
