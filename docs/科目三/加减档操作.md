# 加减档操作

* 上坡、下坡、加速减速全都用抵挡，高档只是为了维持高速运行而已
* 全程不可以用空档滑行
* 尽量使用能够满足当前行驶速度及加速度的最高档位
* 档位越低发动机功率越高，扭矩越强，车加速越快。需要加速度越大约需抵挡
* 六档加速会非常慢，这时候用抵挡加速，高档维持速度。低档加速高档维速
* 最科学的换挡应该看转速表而不是时速表
