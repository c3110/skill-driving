# 超车

1. 必须打左转向灯超过3s，不然直接挂掉
2. 确认安全后才能打方向盘，方向盘左打不超过45度
3. 不具备超车条件时候不得超车
4. 严禁右侧超车，听到超车你就打左转向灯
5. 超车的时候骚微提速一下下
